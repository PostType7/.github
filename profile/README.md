### Install PostType7 starter

- Clone empty next.js repo with typescript and tailwind<br/>
`git clone git@github.com:PostType7/NextStarterPack.git <your-project-name>`<br/>
- Clone native UI components<br/>
`git clone git@github.com:PostType7/NativeTheme.git <your-project-name>/src/components/themes/NativeTheme`<br/>
- Clone PostType7 basic theme (PureBaldrTheme)<br/>
`git clone git@github.com:PostType7/NativeTheme.git <your-project-name>/src/components/themes/PureBaldrTheme`<br/>
- Clone example PureBaldrTheme page<br/>
`git clone git@github.com:PostType7/NativeTheme.git <your-project-name>/src/pages/PureBaldr-example-backOffice`<br/>
- Go to your projectr `cd/<your-project-name>` install libraries `yarn` and run `yarn dev`
